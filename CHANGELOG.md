# 更新日志

所有 notable 变更都会记录在这个文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2024-03-17

### ✨ 新增
- 实时数据监控：粉丝数、获赞数、关注数
- 消息通知：显示评论回复和@提及
- 数据变化追踪：从启动开始计算增长
- 头像显示：自动加载B站头像（圆形裁剪）
- 一键跳转：点击消息卡片跳转到对应视频
- 窗口置顶：可设置窗口始终置顶
- B站风格UI：采用B站经典粉色主题
- Cookie设置：支持手动输入SESSDATA
- 自动刷新：数据每分钟刷新，消息每2分钟刷新

### 🛠️ 技术实现
- 使用Python + Tkinter构建GUI
- 使用PIL处理头像图片（圆形裁剪）
- 调用B站公开API获取数据
- 多线程刷新避免UI卡顿

### 📦 打包
- 支持打包为单文件EXE
- 支持制作Windows安装包
